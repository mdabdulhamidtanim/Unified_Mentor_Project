# Image Slider

## Project Description
This is an interactive image slider built using HTML, CSS, and JavaScript. It allows users to navigate through images using navigation buttons and includes an automatic slideshow feature.

## Features
- Display a series of images in a responsive slider.
- Navigation buttons to go to the previous and next images.
- Automatic slideshow transitioning every 3 seconds.
- Smooth and visually appealing animations.

## How to Run
1. Download or clone this repository.
2. Open `index.html` in your web browser to view the slider.

## File Structure
- `index.html`: The main HTML file containing the slider structure.
- `styles.css`: The CSS file for styling the slider.
- `script.js`: The JavaScript file for functionality.
- `images/`: Folder containing 5 sample images.
